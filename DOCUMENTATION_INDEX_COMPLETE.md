## 📋 The Issue & Solution

### Issue
All user accounts show the same name "Vennela" because:
- Multiple users registered with the same email
- MongoDB enforces unique emails
- Only one account was actually created

### Solution
Use **different email addresses** for each user role:
- Admin: `vennela79@gmail.com`
- Farmer: `farmer.raj@gmail.com`
- Customer: `customer.priya@gmail.com`
- Dealer: `dealer.sai@gmail.com`
