## 🎮 How to Use

### For Dealers:
1. Dashboard → Click "📍 Service Areas"
2. Select District (dropdown)
3. Select Mandal (dropdown)
4. Click "Add as Pickup" or "Add as Drop"
5. Repeat to add more locations
6. Use search to filter if needed
7. Click "Save"
8. Done!

### Result:
- Locations saved to profile
- Customers see you only in your service areas
- More accurate request matching
